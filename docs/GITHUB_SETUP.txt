# GitHub Setup Instructions

After creating your repository on GitHub, run these commands:

# Add your GitHub repository as remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/sentiment-analysis-dashboard.git

# Rename branch to main (if needed)
git branch -M main

# Push to GitHub
git push -u origin main

# Example with actual username:
# git remote add origin https://github.com/johndoe/sentiment-analysis-dashboard.git
# git branch -M main
# git push -u origin main

# You'll be prompted for your GitHub credentials
# Use a Personal Access Token instead of password if required
